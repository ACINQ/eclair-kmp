INSTALLATION:
1. Install Docker for MacOS: https://www.docker.com/get-started
2. Install XQuartz: brew cask install xquartz
3. Start XQuartz (in Applications / Utilities)
4. In XQuartz > Preferences... > Security, enable "Allow connections from network clients"
5. Restart your mac

RUN:
1. Start XQuartz
2. Run ./env.sh clean-run
